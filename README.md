# Simple Log Risk Analyzer

A lightweight Python tool to analyze log files and detect suspicious entries.

## Features
- Keyword-based risk detection
- Risk ratio calculation
- Clean output report
- No dependencies

### Output Formats

- txt : Human-readable report
- json : Machine-readable structured output
- csv : Spreadsheet-friendly format

Example:
```bash
python main.py -i input.txt -o report.json -f json
